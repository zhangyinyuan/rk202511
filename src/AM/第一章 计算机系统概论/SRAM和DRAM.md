# SRAM和DRAM

## 对比表

| 特性             | **SRAM（静态 RAM）**                      | **DRAM（动态 RAM）**          |
| ---------------- | ----------------------------------------- | ----------------------------- |
| **存储单元结构** | 6 个晶体管（6T）                          | 1 个晶体管 + 1 个电容（1T1C） |
| **速度**         | 快                                        | 较慢                          |
| **功耗**         | 高（即使不操作，6个晶体管耗电也比DRAM大） | 低                            |
| **集成率**       | 低（单元大，能集成的少）                  | 高（单元小，能集成的多）      |
| **容量**         | 小                                        | 大                            |
| **成本**         | 高                                        | 低                            |
| **是否需要刷新** | 否                                        | 是                            |
| **主要用途**     | CPU Cache（L1/L2/L3）                     | 主存（内存条）                |
